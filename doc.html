KottValidate

Perform easy, fast and reliable validation on HTML forms

# Introduction

KottValidate is a jQuery form validation plugin that makes form validation easy and stress free.

# Installation

To make use of KottValidate, include KottValidate's compiled and minifed css and JS in your project. KottValidate makes use of bootstrap 5.

Or

Download the source files

You can make changes to the styling of the feedback messages and icons by downloading the KottValidate source files. Click the download button below to download the source files.

# Options for the validate form method and values

Tap on an option to see a demo code on how to correctly use it.

# Name Default Value Description
optionalFields [] An array of CSS id selector of input fields that are optional and ignores them in the validation
Demo code: 
$("#form").validateForm({
         optionalFields:["#otherNames", "#dob"]
    })

Here, the input fields with id's otherNames and dob will be ignored in the validation.
checkPasswordStrength false When set to true checks the strength of input field ":password"
Demo code: 
$("#form").validateForm({
         checkPasswordStrength:true
    })

Here, the values of input fields with password type will be tested against regex values which is dependent on the value of the passwordStrict value.
passwordStrict false When set to true accepts only strong password otherwise, a medium password would be accepted too.
Demo code: 
$("#form").validateForm({
         checkPasswordStrength: true,
         passwordStrict: true
    })

Strong password: Password must have at least 8 characters and contain at least 1 lowercase, 1 uppercase and 1 special character.

Medium password: Password must have at least 6 characters and contain at least one lowercase and one uppercase character or has at least one lowercase and one numeric character or has at least one uppercase and one numeric character.
confirmPassword [] Accepts two css id selector of two password input fields. Checks if the values of both input fields are same and if not, displays and error message
Demo code: 
$("#form").validateForm({
        confirmPassword: ["#password1", "password2"]
    })
customFeedbackMsg {} An object with css selectors (id, name or type) as object keys and custom feedback message as values.
Note: id > name > type
Demo code: 
$("#form").validateForm({
        customFeedbackMsg:{
            ":name":"Please enter your full name",
            "#email": "Please enter a valid email address"
        }
    })
complete A function to that should run on successful form validation.
Demo code: 
$("#form").validateForm({
        complete: function(){
            $("body").empty()
        }
    })

Note: Using appropriate type attributes for input fields enhances form validation.
